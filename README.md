# Multi-Class Face Recognition System

A real-time face recognition and identification pipeline built using **TensorFlow** and **Keras**. The system leverages a Deep Convolutional Neural Network (CNN) to detect human faces and perform multi-class classification to instantly identify specific individuals.

---

## ✨ Features

* **Real-Time Identification:** Captures video or image streams to recognize multiple trained faces instantly.
* **Multi-Class Classification:** Goes beyond simple detection to classify and match faces to specific individuals in the training dataset.
* **Deep Learning Backbone:** Built on a Convolutional Neural Network (CNN) optimized for feature extraction and high classification accuracy.

---

## 🛠️ Tech Stack

* **Framework:** TensorFlow & Keras
* **Language:** Python
* **Libraries:** OpenCV (for image processing/camera feed), NumPy
